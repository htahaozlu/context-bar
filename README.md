# Zed Context Pilot

`Zed Context Pilot`, Zed Assistant paneline proje durumunu yüksek sinyalli bir briefing olarak enjekte eden bir Zed extension prototipidir.

İlk milestone:

- Zed extension iskeleti
- `/hello` slash command
- doğrulanmış araştırma notları
- sonraki fazlar için mimari plan

Dokümanlar:

- [docs/01-research.md](docs/01-research.md)
- [docs/02-architecture.md](docs/02-architecture.md)

## Geliştirme

Zed içinde dev extension olarak yüklemek için bu klasörü seç:

1. `cmd-shift-x`
2. `Install Dev Extension`
3. bu dizini seç

Debug için:

- `zed: open log`
- terminalden `zed --foreground`
