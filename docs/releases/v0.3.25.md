# ContextBar v0.3.25

## Changed

- **Cost tab — full table, ccusage parity.** The per-day breakdown is now a native, column-aligned table showing the complete token split — **Input · Output · Cache+ · Cache↻ · Total · Cost** — grouped by day with per-project sub-rows and a grand Total, mirroring `ccusage daily` but rendered natively (tabular figures, right-aligned numbers, native padding and colors). It keeps what a CLI can't: the monthly plan-value projection, the cache-savings line, and the interactive hover trend.
- **Settings, simplified the Apple way.** Nine tabs became a clean set: the three data views (Usage · Stats · Cost) plus three consolidated preference panes — **General** (menu-bar title, display behavior, and notifications, which used to be three separate tabs), **Appearance**, and **Privacy** — and About. No more overlapping "Menubar / Display / Appearance" tabs to guess between.
