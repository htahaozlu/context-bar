# ContextBar v0.1.2

Universal macOS binary, self-regenerating HUD, and app icon.

## Highlights

- **Universal macOS DMG**: single `ContextBar.dmg` now runs natively on Apple Silicon **and** Intel Macs (no Rosetta).
- **Self-regenerating HUD**: the menubar app bundles the Rust engine and watches the repo via FSEvents, so `hud.json` stays fresh without an external daemon.
- **App icon**: bundle now ships with a proper `AppIcon.icns`, so Finder, Dock, and Launchpad render the ContextBar logo instead of the generic placeholder.

## Installation

1. Download `ContextBar.dmg` from the release assets (universal: Apple Silicon + Intel).
2. Drag `ContextBar.app` into `Applications`.
3. Launch from `Applications`.

## Verify architecture

```bash
lipo -info /Applications/ContextBar.app/Contents/MacOS/context-bar
# expected: x86_64 arm64
```

## Notes

- Minimum macOS: 13.0.
- DMG is Developer ID signed and notarized.
- Existing v0.1.1 users can drag-replace the app in `/Applications`.
