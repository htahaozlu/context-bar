# ContextBar v0.5.0

## Added

- **Pure-Rust engine + cross-platform CLI binaries.** The native engine is now pure Rust — no `python3` anywhere on the hot path. `usage_signal.py` was fully ported to Rust (pricing, aggregate, collect, online, and friends), and `collect_native()` no longer spawns a Python interpreter. That made musl and Windows targets viable, so every GitHub release now ships **6 prebuilt binaries**: macOS arm64/x64, Linux musl arm64/x64, and Windows arm64/x64. Grab one and run.
- **`cargo install context-bar`.** The engine is self-contained Rust now, so a single `cargo install` gets you the full CLI with no separate checkout, no embedded interpreter, nothing to materialize.

## Changed

- **The port is verified, not vibes.** The Rust engine is validated against the Python original three ways: golden fixtures (the 488-row pricing table matches byte-for-byte; aggregate output matches field-for-field), two independent parity reviews, and a real-data differential — Python vs. Rust over the same frozen `~/.claude` + `~/.codex` trees, field-for-field match.
- **Live pricing carries over.** `pricing.rs` fetches the LiteLLM table over HTTP with a 24h on-disk cache and a bundled fallback — same strategy as before, same cost fidelity. Figures are still API-equivalent estimates, not a bill.
- **Online overlays ported, cross-platform.** Statusline snapshot, the Anthropic usage API (account 5h/7d %), and Codex transcript rate-limits all moved to Rust, with cross-platform credential reads (`~/.claude/.credentials.json`; macOS keychain).
- **Nothing changes for menubar users.** The macOS menubar app and the Homebrew cask look and behave exactly as before — the bundled engine is just faster and self-contained. The terminal CLI (`daily / weekly / monthly / session`) from 0.4.0 is unchanged. The `.py` survives only to power the wasm Zed extension.

## For developers

- **`cargo install context-bar`** — self-contained, no Python.
- **Download a binary** — six targets attached to every [GitHub release](https://github.com/ozlu/context-bar/releases): macOS arm64/x64, Linux musl arm64/x64, Windows arm64/x64.
- **`npx context-bar`** — packaging is documented and on the way; the npm publish is pending maintainer credentials.
