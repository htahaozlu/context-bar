# ContextHUD v0.1.8

Agent brand icons across the menubar surfaces.

## Highlights

- **Real provider icons**: supported agents now use bundled brand assets instead of text-only labels or placeholder symbols. Claude uses Anthropic branding, Codex uses the OpenAI blossom, and supported secondary tools such as Gemini, Copilot CLI, and Ollama follow the same rule.
- **Consistent UI**: the same icon treatment now appears in Settings preview, the menubar title, dropdown rows, and the native usage panel.
- **Release-safe assets**: brand icons are copied into the app bundle during packaging so local builds and tagged releases render the same UI.
