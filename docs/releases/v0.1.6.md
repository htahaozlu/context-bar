# ContextBar v0.1.6

Smoother in-app updates.

## Highlights

- **Update in place**: `Check for Updates` now downloads the release DMG inside the app instead of bouncing users to GitHub or leaving files in Downloads.
- **Restart to Update**: once the new build is ready, ContextBar prompts the user to restart and completes the install flow automatically.
- **More native UX**: release notes are still available, but the primary path is a direct desktop-style update experience.
