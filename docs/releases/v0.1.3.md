# ContextHUD v0.1.3

Menubar polish, smarter reset timers, and Homebrew Cask distribution.

## Highlights

- **Cleaner menubar header**: dropdown header no longer shows the version line. Version still visible in Settings → About.
- **Day-aware reset timers**: long reset windows now render as `6d 3h` instead of `149h`. Localized units in Turkish (`6g 3sa`).
- **Homebrew Cask**: install and upgrade via Homebrew:

  ```bash
  brew tap htahaozlu/context-hud https://github.com/htahaozlu/context-hud.git
  brew install --cask context-hud
  ```

  Subsequent releases bump the Cask automatically from the release workflow.

## Installation

1. `brew install --cask context-hud`, or
2. Download `ContextHUD.dmg` from the release assets and drag into `Applications`.

## Notes

- Minimum macOS: 13.0.
- DMG is Developer ID signed and notarized.
- Cask `sha256` is rewritten by the release workflow after the DMG is built.
