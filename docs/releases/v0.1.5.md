# ContextHUD v0.1.5

Native Claude context parity and Turkish localization cleanup.

## Highlights

- **Claude Code parity**: ContextHUD can now ingest Claude Code's native statusline payload through `context-hud claude-statusline` and use it as the primary source for Claude context percentage.
- **Safer fallback**: when the native snapshot is missing, transcript-based estimation now recognizes 1M-context Claude models like Sonnet 4.6 and Opus 4.6/4.7.
- **Turkish text fixes**: corrected missing Turkish characters in the native macOS Settings and About UI.

## Claude Code setup

Add this to Claude Code settings to persist the native snapshot locally:

```json
{
  "statusLine": {
    "type": "command",
    "command": "context-hud claude-statusline"
  }
}
```

This writes `~/.context-hud/claude-statusline.json`. ContextHUD reads that file first, then falls back to transcript estimation only if the snapshot is missing or stale.
