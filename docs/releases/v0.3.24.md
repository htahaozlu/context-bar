# ContextBar v0.3.24

## Fixed

- Cost trend "peak" label no longer stacks into overlapping text on repeated reloads.

## Added

- **Interactive cost trend** — hover any day in the 30-day chart to read its date, estimated cost, and tokens (crosshair + tooltip). Replaces the static sparkline.
- **Session cost in the menubar popover** — the active session now shows its estimated API-equivalent cost next to context usage, so a glance answers "how much did this session cost?"

## Changed

- The Cost tab now spells out the estimate caveat precisely: Claude/Codex transcripts don't record whether a session ran on a subscription or an API key, so all cost is shown as an estimate. Subscription usage isn't billed per token; API-key usage is — the logs can't separate them.
